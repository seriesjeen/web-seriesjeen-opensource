# 🎬 SeriesJeen OOP Client
**ระบบเว็บไคลเอนต์เครื่องเล่นภาพยนตร์ซีรีส์อัจฉริยะ (PHP OOP MVC Web Client)**

เว็บแอปพลิเคชันสำหรับสตรีมมิ่งซีรีส์คุณภาพสูง เชื่อมโยงกับ Series Open API (`api.seriesjeen.online`) พัฒนาด้วยสถาปัตยกรรม **Object-Oriented Programming (OOP) MVC** ที่มีความปลอดภัยระดับพรีเมียม สวยงามสไตล์ **Cyberpunk Glassmorphism** และรองรับระบบลื่นไหลไร้รอยต่อในระดับแอปพลิเคชันหน้าเดียว (SPA)

## ✨ คุณสมบัติเด่นของระบบ (Key Features)

### 📹 ระบบเครื่องเล่นและการสตรีมมิ่งขั้นสูง
*   **ระบบสตรีมคู่ (HLS & Melolo Decryption)**: รองรับการเล่นสตรีมมิ่งธรรมดาผ่านไฟล์ `.m3u8` (HLS.js / Native) และการดาวน์โหลดไฟล์เข้ารหัส **CENC-encrypted MP4 (Melolo)** นำมาถอดรหัสในเบราว์เซอร์ผ่าน WebCrypto (AES-128-CTR)
*   **สลับตอนข้ามฟากไร้รอยต่อคงสถานะเต็มจอ (Seamless Fullscreen SPA Transition)**: ใช้กลไก Fetch และ History API เปลี่ยนเนื้อหาตอนถัดไป ป้ายกำกับข้อมูล เมนู และ URL บน Address bar โดยที่ผู้เล่น**ไม่หลุดออกจากโหมดขยายหน้าจอเต็มจอ (Fullscreen)** และวิดีโอตอนถัดไปเล่นอัตโนมัติทันที
*   **ป้องกันการค้างด้วย AbortController**: ทุกครั้งที่สลับตอนหรือกดปิด/รีเฟรชหน้าเว็บ ระบบจะยิงคำสั่ง `.abort()` ทำลายการเชื่อมต่อดาวน์โหลดตอนเก่าระดับ Network ทันที ช่วยเพิ่มความเร็ว คืนแบนด์วิดท์ และแก้ปัญหาบราวเซอร์ล็อกพอร์ต Timeout
*   **การ์ดนับถอยหลังกระจกเบลอ (Autoplay Overlay)**: เมื่อเล่นวิดีโอใกล้จบ (12 วินาทีสุดท้าย) จะปรากฏหน้าจอนับถอยหลังถัดไปพร้อมอนิเมชันวงกลม SVG หรูหรา สั่งเล่นข้ามตอนได้ทันที หรือกดกดยกเลิกเพื่อดูต่อจนจบปกติ

### 🔒 มาตรการความปลอดภัยและป้องกันลิขสิทธิ์
*   **การดักดีบัคค้างถาวร (Debugger Loop Trap)**: ติดตั้งกับดักป้องกันผู้ใช้เปิดแผงนักพัฒนา (DevTools) ทุกช่องทางด้วยการวนลูปคำสั่ง `debugger;` แบบไดนามิกผ่านโครงสร้าง `.constructor("debugger").call()` ทำให้บราวเซอร์หยุดทำงานค้างถาวรทันทีหากพยายามเปิดตรวจสอบโค้ดเพื่อดูดลิงก์วิดีโอ (สคริปต์นี้จะข้ามหน้าควบคุมของแอดมินหลังบ้าน `/admin` เพื่อให้แอดมินทำงานดีบัคหน้าจอควบคุมได้สะดวกปกติ)
*   **บล็อกการสกัดซอร์สโค้ดฝั่งหน้าบ้าน**: บล็อกปุ่มลัด **F12**, คีย์ลัดชุดคำสั่ง DevTools ทุกตัวบนคอมพิวเตอร์ (**Ctrl+Shift+I / J / C** และ **Cmd+Opt+I / J / C**), ปุ่มลัดเรียกดูซอร์สโค้ดดิบ (**Ctrl+U** / **Cmd+U**), และการสั่งเซฟหน้าเว็บ (**Ctrl+S** / **Cmd+S**) รวมถึงบล็อกการคลิกขวาเมาส์ทั่วเว็บไซต์
*   **ระบบล็อกอุปกรณ์รายเครื่อง (Web-HWID Locking)**: จำกัดการใช้งานโค้ดกุญแจเข้าชม 1 คีย์รหัสต่อ 1 เครื่อง/เบราว์เซอร์สำหรับสมาชิกทั่วไป โดยการบันทึก IP ลายพิมพ์เว็บ (User-Agent) และคุกกี้จำเพาะHttpOnly ป้องกันการแบ่งปันคีย์ข้ามระบบปฏิบัติการหรือก๊อปปี้คุกกี้

### 🎨 แผงควบคุมผู้ดูแลระบบหลังบ้านระดับพรีเมียม (Admin Dashboard)
*   **ระบบสุ่มรหัสกุญแจ UUID v7**: สร้างคีย์รหัสผ่านเข้าชมสไตล์ UUID v7 ที่ปลอดภัย แข็งแกร่ง และเรียงตามลำดับเวลาเพื่อประสิทธิภาพของดัชนีฐานข้อมูล สามารถกำหนดประเภทสิทธิ์ วันหมดอายุ (หรือระบุวันเวลาตามใจชอบผ่านปฏิทิน Custom datetime)
*   **ปุ่มคัดลอกกุญแจพร้อมอนิเมชัน**: คัดลอกรหัส UUID ลงคลิปบอร์ดในคลิกเดียวพร้อมกล่อง Toast SweetAlert และอนิเมชันปุ่มติ๊กถูกสีเขียวมรกตฟีดแบคทันที
*   **มาตรการล็อกป้องกันตัวเอง (Lockout Prevention)**: บล็อกการลบคีย์แอดมินหลักรหัสสุดท้าย เพื่อการันตีว่าแอดมินจะไม่มีวันล็อคขังตัวเองออกนอกระบบหลังบ้านอย่างถาวร
*   **ระบบตั้งค่าเว็บไซต์ไดนามิก**: ควบคุมชื่อเว็บ, อัปโหลดโลโก้ (Fallback เป็นไอคอนอัตโนมัติ), ลิงก์ Footer, และคำอธิบายหน้าล็อกอินได้อย่างยืดหยุ่น
*   **ระบบสีธีมและเลย์เอาต์ (Website Styling Customizer)**: กำหนดสีหลักของแบรนด์, สีเฉดคู่ไล่สีนีออน, สีแถบนำทาง (Navbar), สีพื้นหลัง (Body), และส่วนล่าง (Footer) ได้อิสระ พร้อมระบบ **Live Preview ย่อส่วนเสมือนจริง** ในหน้าตั้งค่าหลังบ้าน แสดงผลตอบรับทุกเฉดสี Hex ทันทีขณะเลือกลาก
*   **ตารางสถิติผู้ใช้งาน**: ติดตั้งเครื่องมือ DataTables ไฮบริดสี Dark Cyberpunk เรืองแสงภาษาไทย จัดการค้นหา สถิติ IP, ล็อกอุปกรณ์ และเวลาเปิดใช้งานล่าสุด พร้อมปุ่มกดล้างล็อกอุปกรณ์ (Reset HWID) ในคลิกเดียว

### 404 Error Page ดีไซน์ใหม่
*   หน้าจอข้อผิดพลาด 404 แบบแก้วกึ่งโปร่งใส ล้อมรอบด้วยโคมไฟนีออนเบลอเรืองแสงกระพริบช้า ๆ ตัวเลขยักษ์ฉาบเฉดสีรุ้งไล่ระดับ พร้อมปุ่มป้อนคำสั่งย้อนกลับประวัติเว็บลดแรงต้านการใช้งานอย่างหรูหรา

## 🛠️ ความต้องการของระบบ (System Requirements)

*   **PHP**: เวอร์ชัน `8.1` หรือสูงกว่า (แนะนำ `PHP 8.2` ขึ้นไปจนถึง `PHP 8.5` รองรับได้อย่างเสถียร)
*   **MySQL / MariaDB**: ฐานข้อมูลหลักสำหรับการบันทึกสิทธิ์และตั้งค่าเว็บไซต์
*   **Composer**: เครื่องมือจัดการแพ็กเกจไลบรารีของ PHP
*   **Web Server**: Apache ที่เปิดใช้งาน Mod_Rewrite หรือ Nginx (หรือใช้เซิร์ฟเวอร์จำลอง PHP)

## ⚙️ ขั้นตอนการติดตั้งและการเริ่มใช้งาน (Setup & Installation)

### 1. โคลนและดาวน์โหลดไฟล์โปรเจกต์
ดาวน์โหลดซอร์สโค้ดโปรเจกต์ลงบนพื้นที่ไดเรกทอรีเว็บไซต์ของคุณ (เช่น `/var/www/html` หรือโฟลเดอร์ทำงานของเครื่องจำลอง)

### 2. ติดตั้งส่วนขยายผ่าน Composer
เปิดโปรแกรม Terminal หรือ Command Prompt ในโฟลเดอร์โปรเจกต์หลัก และรันคำสั่ง:
```bash
composer install
```
*ระบบจะดาวน์โหลดไลบรารี `guzzlehttp/guzzle` และ `vlucas/phpdotenv` มาติดตั้งในโฟลเดอร์ `vendor`*

### 3. ตั้งค่าสภาพแวดล้อมสิทธิ์การใช้งาน
คัดลอกไฟล์ต้นแบบการตั้งค่าค่าเริ่มต้นของระบบ:
```bash
cp .env.example .env
```
เปิดไฟล์ `.env` ที่สร้างขึ้นใหม่ด้วยโปรแกรม Text Editor และระบุรายละเอียดฐานข้อมูลและ Private Key หลักหลังบ้าน:
```env
# ตั้งค่าฐานข้อมูลหลัก MySQL
DB_HOST=
DB_PORT=3306
DB_NAME=
DB_USER=root
DB_PASS=รหัสผ่านฐานข้อมูลของคุณ

# คีย์ถอดรหัส APIs ส่วนตัวของ SeriesJeen สำหรับการดึงรายการแพลตฟอร์ม
SERIES_API_KEY=private_series_api_key_here
```

### 4. เรียกทำงานผ่าน PHP Built-in Server
ในกรณีที่ต้องการรันทดสอบบนเครื่อง Local ของคุณอย่างรวดเร็ว สามารถรันระบบผ่านคำสั่ง:
```bash
php -S localhost:8106 router.php
```
จากนั้น เปิดโปรแกรมเว็บบราวเซอร์ของคุณแล้วเข้าชมที่ที่อยู่: **`http://localhost:8106`**

## 📁 โครงสร้างโปรเจกต์หลัก (Project Architecture)

*   `src/Core/App.php` : ไฟล์ตัวประคองแอปพลิเคชัน โหลดตัวแปรและบูสระบบความปลอดภัยหลัก
*   `src/Core/Database.php` : ไฟล์ระบบคลาสเชื่อมต่อ PDO MySQL พร้อมความสามารถ **Auto-Migration** ที่จะสร้างตาราง `key_codes` และ `settings` เติมข้อมูลเริ่มต้นให้อัตโนมัติเมื่อรันเว็บครั้งแรก
*   `src/Core/Router.php` : คลาสตัวกระจายพาธ (Dispatcher) วิเคราะห์ลิงก์คำขอ สกรีนตัวกรองมิดเดิลแวร์ และส่งต่อตัวแทนคอนโทรลเลอร์
*   `src/Controllers/` : กลุ่มคอนโทรลเลอร์จัดการทราฟฟิก (หน้าบ้าน, หน้าแอดมิน, ล็อกอิน/ล็อกเอาต์)
*   `views/` : โฟลเดอร์ที่เก็บหน้าจอแสดงผล HTML/PHP แยกย่อยเป็นสัดส่วน (Layouts, Partials, Admin Panels, Errors 404/500)
*   `public/assets/` : พื้นที่จัดเก็บไฟล์สื่อสารหลักฝั่งไคลเอนต์
    *   `js/player.js` : ตัวควบคุมระบบเครื่องเล่นอัจฉริยะ (SPA Transition, Abort Controller Network Release, SVG countdown)
    *   `js/melolo.js` : ตัวคอมไพล์สัญญาสตรีมดึงกุญแจ DRM และถอดรหัสในเบราว์เซอร์
    *   `css/style.css` : กฎ CSS Overrides สำหรับ Plymouth Controls บนสมาร์ตโฟนโมเดิร์น
